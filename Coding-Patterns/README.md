# 🧠 Coding Interview Prep — By Pattern

A curated set of **30 problems** covering 80% of coding interviews, organized by pattern.
Each problem has solutions in both **Python** and **C#**.

## 📁 Structure

```
interview-prep/
├── 01_arrays_strings/
├── 02_trees/
├── 03_graphs/
├── 04_dynamic_programming/
├── 05_intervals_greedy/
├── 06_stack_queue/
├── 07_binary_search/
└── 08_linked_lists/
```

Each folder contains:
- `README.md` — pattern overview + problem list
- `<problem_name>.py` — Python solution with explanation
- `<problem_name>.cs` — C# solution with explanation

## 🏁 8-Week Study Plan

| Week | Focus | Problems |
|------|-------|---------|
| 1–2 | Warm-up (no timer) | Arrays/Strings, Linked Lists |
| 3–4 | Pattern drilling (35 min/problem) | Trees, Graphs, Stack/Queue |
| 5–6 | Speed + pressure (25 min hard timer) | DP, Binary Search, Intervals |
| 7–8 | Mock interviews | Full rotation, all patterns |
